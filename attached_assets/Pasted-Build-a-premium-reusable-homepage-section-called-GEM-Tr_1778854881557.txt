Build a premium reusable homepage section called “GEM Trust Network VideoFlow” for the new GEM website.

This section must show a slow, steady, right-to-left moving horizontal strip of trustee and partner cards. It should feel like a cinematic video-flow brand wall, not a fast carousel or random logo slider.

The purpose is to visually communicate GEM’s credibility network: trustees people can contact/call, plus business allies and technology partners that support GEM’s cybersecurity, asset recovery, real estate, legal/trust, investment, cloud, and financial infrastructure services.

SECTION TITLE:
Our Trusted Partners & Business Allies

SECTION SUBTITLE:
Working with world-class organizations, trustees, and strategic allies to deliver excellence.

PLACEMENT:
Place this section near the top of the homepage, preferably directly after the hero section and before the core services section. Also make the component reusable for the existing Partners & Trustees page.

COMPONENT NAME:
PartnersTrusteesVideoFlow

VISUAL STYLE:
Use a premium dark corporate style aligned with GEM’s brand:
- Dark navy / black background
- Gold accents for trustees
- Blue / silver accents for partners
- Glassmorphism-style cards
- Subtle glow
- Rounded cards
- Professional enterprise look
- Smooth, stable, executive motion

MOTION REQUIREMENT:
The strip must move continuously from right to left.
It must be slow, steady, and premium, similar to the previous website’s gentle moving logo strip.

Animation behavior:
- Direction: right to left
- Speed: slow, around 38–55 seconds per full loop
- Infinite loop
- No blank space
- No visual jump
- No flicker
- No hard reset
- Desktop: pause animation on hover
- Mobile: keep animation moving continuously
- Use duplicated track/list for seamless looping

Do not build this as a static section.
Do not build this as a manual click slider.
Do not use arrows as the main interaction.
This should behave like a passive animated VideoFlow.

CARD TYPES:
The data must support three types:
1. trustee
2. partner
3. business-ally

Trustee cards must be visually stronger than partner cards because users may need to call or contact GEM’s trustees.

TRUSTEE CARD STYLE:
- Gold border
- Gold TRUSTEE badge
- Slightly stronger glow
- Optional CTA such as “View Trustee” or “Call Trustee”
- Trustee cards should stand out clearly inside the moving strip

PARTNER CARD STYLE:
- Blue, silver, or neutral border
- PARTNER / BUSINESS ALLY / TECH PARTNER badge
- Logo or placeholder icon
- Segment label
- Role description
- Optional “View Partner” CTA

DATA STRUCTURE:
Create a data-driven array so future updates only require editing the data, not the component layout.

Use this structure:

{
  id: string,
  type: "trustee" | "partner" | "business-ally",
  name: string,
  segment: string,
  role: string,
  logo: string,
  link?: string,
  cta?: string,
  displayStatus?: "approved" | "example" | "placeholder"
}

IMPLEMENTATION APPROACH:
Use custom CSS animation with duplicated items for a seamless marquee loop.

Recommended structure:
<section className="partners-video-flow">
  <div className="section-header">
    <span>GEM Trust Network</span>
    <h2>Our Trusted Partners & Business Allies</h2>
    <p>Working with world-class organizations, trustees, and strategic allies to deliver excellence.</p>
  </div>

  <div className="video-flow-mask">
    <div className="video-flow-track">
      Render all cards once
      Render all cards again for seamless loop
    </div>
  </div>
</section>

CSS behavior:
- .video-flow-mask uses overflow: hidden
- .video-flow-track uses display: flex
- .video-flow-track width should be max-content
- Animate transform from translateX(0) to translateX(-50%)
- Duplicate the data array in render
- Add gradient fade on left and right edges
- Pause animation on hover only for desktop

RESPONSIVE REQUIREMENTS:
Desktop:
- Multiple cards visible at once
- Cards around 270–320px wide
- Slow motion, premium spacing

Tablet:
- Cards around 240–280px wide
- Maintain readability

Mobile:
- Cards around 220–250px wide
- Animation keeps moving
- No hover pause
- Text remains readable
- Section should not cause horizontal page overflow

ACCESSIBILITY:
- Use semantic section markup
- Add aria-label for the carousel region
- Logo images must have alt text
- Links must be keyboard accessible
- Respect reduced motion where possible by slowing or disabling animation for users with prefers-reduced-motion
